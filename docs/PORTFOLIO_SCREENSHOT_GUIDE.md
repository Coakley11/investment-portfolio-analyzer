# Portfolio Screenshot Guide — Investment Portfolio Analyzer

## Enable Screenshot Mode

1. Sidebar → **Portfolio Screenshot Mode**
2. Load a preset (Balanced or Aggressive) for visual richness
3. Open **Portfolio Analytics** tab (Advanced) or **④ Analyze Portfolio** (Beginner)

---

## Screenshot Mode Behavior

- Replaces full hero badges with compact hero banner
- Executive summary card at top of Analytics tab
- Hides sidebar instructional captions
